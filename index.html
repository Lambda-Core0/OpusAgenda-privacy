Política de Privacidad

OpusAgenda funciona íntegramente de forma local en el dispositivo.

Datos del usuario

Esta aplicación no recopila, almacena ni transmite datos personales a servidores externos.

Almacenamiento

Toda la información, incluyendo tareas, categorías, notas, audios y archivos, se guarda únicamente en el dispositivo del usuario mediante almacenamiento local.

Acceso a permisos

La aplicación puede solicitar permisos o accesos locales como:

  • Micrófono, para grabación de notas de voz.
  • Acceso a archivos seleccionados por el usuario, para exportar e importar respaldos.

Estos permisos se utilizan únicamente para funciones internas de la app y no se envía información fuera del dispositivo.

Servicios externos

La aplicación no utiliza servicios de análisis, publicidad ni seguimiento de usuarios.

Contacto

Si tienes dudas sobre esta política, puedes contactar al desarrollador a través del repositorio del proyecto.
